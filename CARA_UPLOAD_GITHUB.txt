## LANGKAH 2: Hubungkan dan Upload
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

Setelah repository dibuat di GitHub, jalankan command berikut 
di terminal (satu per satu):

### 1. Set remote repository
Ganti [URL-GITHUB-ANDA] dengan URL repository Anda

```
git remote add origin [URL-GITHUB-ANDA]
```

Contoh:
```
git remote add origin https://github.com/username/DigiMart.git
```

### 2. Ubah branch name ke main (jika perlu)
```
git branch -M main
```

### 3. Push ke GitHub
```
git push -u origin main
```
